
# E-Ticaret Sitesi

Çevrimiçi ürün satışı yapılan platform.


## Demo

[Demo](https://eticaretsitesi.somee.com/)  

  
## Özellikler

- Asp.Net Core MVC
- Katmanlı Mimari
- Entity Framework
- Entity Framework Identity
- Authentication ve Role-Based Authorization
- Microsoft Sql Veritaban
- Responsive Tasarım
